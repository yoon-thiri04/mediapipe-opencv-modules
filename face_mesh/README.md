# Face Mesh with Media Pipe

This repository contains a Python module for **Face Mesh** using **MediaPipe**. It allows you to detect and extract detailed facial landmarks from faces in images or video streams.  

## Features
- Detects up to **2 faces** at a time.  
- Returns all **468 facial landmarks** for each detected face.  
- Provides **landmark IDs** along with their normalized coordinates.  
- Easy-to-use Python module: `face_mesh_module`.  

## Included Files
- `face_mesh_module.py` – Python module that returns facial landmarks.  
- Sample videos/images – For testing and visualization.
